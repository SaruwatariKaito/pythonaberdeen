Python Edinburgh

The user group for Pythonistas in Edinburgh.

We're a user group providing technical, professional and social support for Python programmers based in Edinburgh. We meet in the pub monthly, and periodically run other events.

When and Where

Python Edinburgh meets on the 4th Tuesday of each month. Other events such as talks and sprints are hosted less frequently.
